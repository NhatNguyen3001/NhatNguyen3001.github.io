Feedback

Feedbackers

Bao

The first feedback I receive from Bao is that the app maybe too complicated for users and security is still a little bit loose. In my opinion, this is a great feedbacks. With many functions in my apps and 2 different UI, many people will find it difficult to fully comprehend all the usages in the app. In order to fix this problem, I will find methods to simplified the app without removing any functions. Moreover, I will design both the UIs to make them user-friendly, structured so that users won’t get confused between functions. As for the the security, with many access in one period of time, the app is prone to attacks. To solve this problems, I will hire Cyber Security specialists to protect the app from any outside attacks.

Long

The second feedbacks from my friends is that with such huge database and many features from an app, it will take a very long time to update or fix the bug if there are any problems. Not all problems can be avoided so back-up plans are necessary. More IT specialists will be hired to keep the system running. We will also frequently ask for feedbacks from customers so that we can detect any potiential issues. The second feedback I receive from Long is that with so many features available in the app, it will drain the battery of your devices. To solve this problem, we will add the “battery-saving mode” into the devices. By turning it on, the app will automically shut down unused features, sligtly reduce brightness.
